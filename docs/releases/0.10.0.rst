RP 0.10.0
=========

This goal of this release is to simplify test data generation step. The
setup code (conftest) must be small as possible:

.. code-block:: python

    import pytest

    import sections
    import power

    power.init(sections.ROOT)

    @pytest.mark.usefixtures('session')
    def pytest_sessionstart():
        power.run()

Accessing the resource(_PDF) and generated resource is very simple:

.. code-block:: python

    BACHELOR111 = power.link(power.BACHELOR111_PDF)

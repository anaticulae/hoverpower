RP 0.12.0
=========

With the next release, we want to support incremental resource
generating. The concept is quite simple:

1. remove generated pdf folder
2. run baw --generate

To archive this simple approach, the power tool must know, which
resources exists and must be available to execute single file generation.

The knowledge about the resources is defined via `RESOURCES` list
variable in conftest.py:

.. code-block:: python

    RESOURCES = [
        (path, ),
        (path, optional:pages),
        ...
    ]

Furthermore you have extend the ``generate`` method:

.. code-block:: python

    def generate(path: str, pages:str):
        pass

To invoke every single resource generation. Without adding parameter to
``generate`` the old single invocation approach is used.
